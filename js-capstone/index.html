Random Student Generator

Each student has a weighted number next to their name which decreases each time they are chosen. The higher the number, the more likely they are to be chosen. A weight of 0 equals no chance of being chosen. If all students end up with a weight of 0 after a student is chosen, their weights will be reset to the default of 3.

You can manually change these values with the + and - buttons next to each student.
